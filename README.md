# Configuration
To run this project, follow these steps:
1. **How to Run**:
   - Install project dependencies:
     ```bash
     bundle install
     ```
   - Upload file .sqlite3 in folder db->development.sqlite3

   - Migrate up DB :
     ```bash
    rails db:migrate
     ```
2. **Start Services**:
   - Migrate up DB :
     ```bash
    Rails server
     ```
